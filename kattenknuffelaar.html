During this project, I:

  • Created an interactive map which allows you to zoom in, explore the world and click on the location markers for information. I used JSON to store the data.

  • Created an interactive customized bar chart with a working filter function and tooltips. I used JSON for the data again.

Check it out!

The process

Firstly, I made a map using the Leaflet.JS library. This was a map showcasing the origin of different cat breeds. It’s interactive, you can zoom in and explore the world, clicking on the different markers for information.

I used a JSON file to store the data and used a loop in JavaScript to create markers for every single cat. I added a custom location marker to make it fit my theme.

When this was done, it was time for my main visualization, a bar chart using D3 showing the most popular cat breeds in different parts of the world. The cats are all custom SVG patterns.

When the bar chart was done, it was time to add a filter function. I had some trouble with it, but with some assistance from my teacher, a working filter function was made. When you clicked one of the buttons, the data and the images change.

The final task was to add a bit of storytelling and make sure everything looked aesthetically pleasing.

The results

Using the knowledge I aqcuired over the past weeks, I made a web page all about cats, with interactive data visualizations.

Kattenknuffelaar

School project • HTML/CSS/JS • Leaflet.JS/D3.JS

The process

Once I had a concept in mind, it was pretty easy to come up with the data visualisations.

I planned to have one main visualization with some additional visuals and information surrounding it.

For the first visualization, I chose to create a map displaying the origins of different cat breeds. I used the Leaflet.js library for this, and after experimenting with it for a while, I successfully created the map. It’s interactive, you can zoom in and explore the world.

The main visualization was going to be a customized bar chart showing the popularity of various cat breeds worldwide. The bars were going to be pictures of the cat breeds, which took quite some time, because they had to be made into custom SVG patterns and I also had to ensure the image would fit inside the bars. This caused me to spend quite some time on Photoshop.

All I had left to do now was to add a filter function, which was a challenge. However, with some assistance, I managed to get it done. I assigned different values to buttons and used an eventListener to ensure that the right items were displayed when clicked.

The final task was to add a bit of storytelling and make sure everything looked aesthetically pleasing.

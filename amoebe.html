INTRODUCTION

During the summer, I worked on the front-end of the system behind the Escaperoom. I didn’t focus too much on the design, but mostly on a good user experience. I worked together with my boyfriend, the back-end developer, to create dynamic content.

PROCESS

When I jumped in to help with this project, a lot had already been build, system wise, which meant there was a lot I could do. I started by designing the layout of the pages and experimenting with colors, I tried to base them on the already existing logo, which contains pastel colors:

I kept the layout pretty simple, since it’s supposed to be easy to navigate and find the stuff you’re looking for. I talked to my friend and my boyfriend about how and where they wanted certain things, since they were going to be the primary users.

RESULTS

Like I said before, this project is still a work in progress. There's a lot that still has to be styled and a lot I want to change, like the light-control you find on the right side of the screen. I wanted this in an individual pop-up, so it wouldn't be on the main page, but my friend really wanted it right there so it's easily accessible. Maybe I'll find a better solution for the way it looks sometime.

Sadly, the Github page is private since there's a lot of personal information like IP-addresses in the back-end code. However, if you decide to invite me for an interview, I could certainly show you some of the code ;)
